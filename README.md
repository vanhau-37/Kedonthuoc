# Kedonthuoc
## các cài đặt
```
Hướng dẫn cài đặt chương trình kê đơn thuốc
B1 : giải nén file CNPM.rar
B2 : Mở xampp và tạo database mới có tên là "donthuoc"
B3 : Up file thuoc.sql lên database mới tạo
B4 : copy folder mới giải nén , dán vào mục htdocs của xampp (C:\xampp\htdocs)
B5 : Run link http://localhost:8080/btweb/CNPM/ 
```
